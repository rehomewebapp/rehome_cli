# rehome_cli
## environment
install with 
```
conda env create -f environment.yml
```

update env
````
conda env update --file environment.yml --prune
````

export env with
````
conda env export > environment.yml
````

## nano for Windows
Umgebungsvariablen > Path > Bearbeiten > Neu > input path to ..rehome_cli/bin/nano.exe
